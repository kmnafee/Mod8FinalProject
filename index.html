Urban Tree Phenophase Explorer

An interactive map of species-level phenophase transitions for urban trees on the University of Kentucky campus in Lexington, Kentucky. Tree polygons are linked to phenology records from 2023–2025 using TARGET_FID.

Method
  • Tree polygons are linked to phenology values using TARGET_FID.
  • Species names are read from the Name field.
  • The right-side panel updates when a tree polygon is clicked.
  • The year selector changes which phenophase values are shown.
